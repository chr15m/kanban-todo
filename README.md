A Kanban board for your Markdown TODO file.

- Drag-and-drop visual editing of plain text Markdown
- Runs 100% in your browser - no server, no database
- Single HTML file - upload to self-host
- CLI tool to launch as a Chrome app: `kbtd TODO.md`

## Format

The text based format is designed to be simple for humans and LLMs to read and modify.

```
# Project name

## Column name

- [ ] Task number one.
- [ ] Some other task.

## Another column

- [ ] Yet another task.
```

## Self host

Copy `index.html` and `kbtd` up to your server.

Then visit the page and download `kbtd` and put it on your path.
