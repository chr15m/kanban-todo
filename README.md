A Kanban editor for your TODO.md Markdown file, in a single `index.html` file.

Launch it from the command line: `kbtd TODO.md`.

It will open Chrome/Edge browser in app/kiosk mode with the running web app.

## Format

```
# Project name

## Column name

- [ ] Task number one.
- [ ] Some other task.

## Another column

- [ ] Yet another task.
```

## Self host

Copy `index.html` and `kbtd` up to your server.

Then visit the page and download `kbtd` and put it on your path.
